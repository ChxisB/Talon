ชุดโครงการชุมชนที่สร้างขึ้นบน OpenCode

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ OpenCode ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-talon](https://github.com/awesome-talon/awesome-talon) และ [talon.cafe](https://talon.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
