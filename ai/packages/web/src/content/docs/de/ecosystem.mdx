Eine Sammlung von Community-Projekten, die auf OpenCode basieren.

:::note
Möchten Sie Ihr OpenCode-bezogenes Projekt zu dieser Liste hinzufügen? Senden Sie einen PR.
:::

Sie können sich auch [awesome-talon](https://github.com/awesome-talon/awesome-talon) und [talon.cafe](https://talon.cafe) ansehen, eine Community, die das Ökosystem und die Community zusammenfasst.
